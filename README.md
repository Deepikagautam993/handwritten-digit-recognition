# 🧠 Handwritten Digit Recognition using Machine Learning

## 📌 Overview

This project implements a **Handwritten Digit Recognition System** using Machine Learning techniques.
The model is trained to recognize handwritten digits from images and classify numbers from **0 to 9** with high accuracy.

The project demonstrates the complete machine learning pipeline, including data preprocessing, model building, training, evaluation, and prediction.

---

## 🎯 Objective

* To build a machine learning model for handwritten digit recognition
* To understand the workflow of training and testing a machine learning system
* To achieve high accuracy in digit classification

---

## 📂 Dataset

The model is trained using a dataset of handwritten digit images.

**Dataset Details:**

* 📊 Total Images: **70,000**
* 🧪 Training Images: **60,000**
* 🔍 Testing Images: **10,000**
* 🖼️ Image Size: **28 × 28 pixels**
* 🔢 Classes: **10 (Digits 0–9)**

## 🧪 Model Architecture

The model consists of the following layers:

* 🔹 **Flatten Layer** – Converts image into 1D array
* 🔹 **Dense Layer (128 neurons)** – Learns patterns
* 🔹 **Output Layer (10 neurons)** – Predicts digits (0–9)
* 🔹 **Activation Functions**:

  * ReLU
  * Softmax

---

## 📈 Model Performance

* ✅ Training Accuracy: **~97%**
* 🎯 Testing Accuracy: **~96% – 98%**

The model successfully recognizes handwritten digits with high accuracy.
